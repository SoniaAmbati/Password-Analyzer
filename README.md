# 🔐 Password Strength Analyzer

A sleek and responsive password strength analyzer built using **HTML**, **CSS**, and **JavaScript**.

## ✨ Features

- Real-time password strength checking
- Dynamic strength meter
- Show / Hide password toggle
- Validation for:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
  - Minimum length
- Clean and modern UI
